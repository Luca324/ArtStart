# ArtStart: Интерактивное приложение для изучения теории цвета

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform: Windows|macOS](https://img.shields.io/badge/Platform-Windows%20%7C%20macOS-blue)]()

**ArtStart** — это десктопное приложение, созданное для начинающих художников и дизайнеров, желающих изучить основы колористики. Проект разработан на языке C# в рамках учебной дисциплины МДК 02.01 "Технология разработки программного обеспечения".

Приложение объединяет в себе три ключевые области: **цветовой круг для смешивания**, **инструмент для рисования** и **викторины для проверки знаний**, предлагая пользователю интерактивную среду для экспериментов с цветом и углубления понимания цветовой теории.

## ✨ Функциональные возможности

Приложение включает в себя три основных модуля, каждый из которых нацелен на развитие определенных навыков:

1.  **Модуль «Смешивание цветов» (Color Mix)**
    *   **Реалистичное смешивание:** Использует специальный алгоритм для имитации смешивания пигментов (как в реальной жизни), а не света (желтый + синий = зеленый).
    *   **Создание палитр:** Пользователь может создавать именованные палитры для организации своих цветов.
    *   **Управление цветами:** Возможность добавлять полученные оттенки в любую из созданных палитр и удалять их.
    *   **Интуитивный интерфейс:** Выбор цветов осуществляется через удобный ColorPicker.

2.  **Модуль «Рисование» (Planned)**
    *   Функционал для создания изображений с использованием цветов из созданных палитр.
    *   Импорт и экспорт изображений в форматах `.png` и `.jpg`.

3.  **Модуль «Обучение и тестирование» (Planned)**
    *   Викторины и тесты для проверки знаний по теории цвета (теплые/холодные цвета, оттенки и т.д.).
    *   Сохранение статистики и отслеживание прогресса пользователя.

4.  **Пользовательская система**
    *   Регистрация и авторизация пользователей.
    *   Личный кабинет с сохраненными палитрами и историей тестирований.

## 🛠️ Технологии и инструменты

*   **Платформа:** .NET (WPF)
*   **Язык:** C#
*   **Сторонние библиотеки:**
    *   `Scrtwpns.Mixbox` - для реалистичного смешивания цветов (пигментная модель).
    *   `Newtonsoft.Json` - для работы с данными пользователей и палитрами.
    *   `Xceed.Wpf.Toolkit` - для элемента выбора цвета (ColorPicker).

## 🚀 Быстрый старт

### Требования к системе
*   **ОС:** Windows 10 или выше, macOS (теоретически, под целевые платформы .NET)
*   **Процессор:** Intel Core i5 и выше
*   **ОЗУ:** 4 ГБ и более

### Установка и запуск
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/artstart.git
    ```
2.  Откройте решение `ArtStart.sln` в Visual Studio.
3.  Восстановите NuGet-пакеты.
4.  Запустите проект (F5).
